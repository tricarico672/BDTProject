# BDTProject

Repository for the Big Data Technologies Project

# Setup

From the root of the directory run `python src/guided_setup.py` to set up the required API keys otherwise some components will not work as intended. For the app to work you need to have access to:

1. A Google Maps API Key for the Distance Matrix API. Follow the official documentation to get one: https://developers.google.com/maps/documentation/distance-matrix/get-api-key
